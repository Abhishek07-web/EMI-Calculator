# EMI Calculator

A clean, free EMI loan calculator built with HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://emi-calculator-chi-three.vercel.app

## Features
- Calculate monthly EMI instantly
- Adjust loan amount, interest rate, and tenure via sliders or input fields
- View complete month-by-month repayment breakdown
- Fully responsive and mobile-friendly
- 100% free — no ads, no paywalls

## How to Use
1. Enter your loan amount
2. Set the annual interest rate
3. Choose loan tenure in months
4. Click **Calculate EMI**

## Tech Stack
- HTML
- CSS
- JavaScript
